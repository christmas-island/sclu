📐 The Formula

SCLU measures any drink relative to a 12oz Coors Light (4.2% ABV).

SCLU = (volume_ml × ABV%) / (355 × 4.2) We always show two values:
SCLU₄.₂ — vs standard Coors Light (4.2%)
SCLU₅.₀ — vs Coors Light Banquet (5.0%)

📊 Reference Points

12oz Coors Light (4.2%) → 1.00 SCLU
16oz Bud Light (4.2%) → 1.33 SCLU
12oz IPA (7.0%) → 1.67 SCLU
750ml Wine (13%) → 7.37 SCLU
1.75L Vodka (40%) → 94.1 SCLU
